# electracars-dev
Book Test Drive Agent
Electra Cars, an original equipment manufacturer (OEM), features a premium website showcasing its latest vehicle models. Currently, customers who want a test drive must fill out a lengthy manual form or request a callback.



📋 Current Pain Points
High Friction: The form is too long, leading to a massive drop-off rate.
Low Conversion: Only 1% of website visitors actually book a test drive.
Manual Bottlenecks: Call center teams spend too much time chasing customers for missing information, such as model, location, and time.
🎯Your Mission
Automate the test drive booking journey. Replace the "clunky form" culture with a seamless, AI-driven experience using Agentforce to maximize conversions and data accuracy.


💡 Sample Scenarios and Requirements
AI-Driven Booking: Use Agentforce to handle the booking conversation, ensuring a natural, low-friction interaction.
Data Capture: Automatically capture the vehicle model, preferred date/time, and zip code during the conversation.
Record Creation: Automatically create the test drive record in Automotive Cloud once the details are confirmed.
Intelligent Routing: Use the captured zip code to assign the lead to the respective dealership.
Omni-channel Reach: Explore how WhatsApp can be used as the primary communication channel instead of a web form.
Confirmation Loop: Notify the customer via their preferred channel once the booking is confirmed.
Proactive Nudge: Brainstorm ways to improve website visitor conversion through AI-led prompts.


🛠️ Technical Requirements
Mandatory Features to be Used

Automotive Cloud: For core data architecture including leads, vehicles, and test drives.
Agentforce: To power the autonomous agent actions and reasoning.
Additional Features

Digital Engagement: For WhatsApp or SMS integration.
Data Cloud: For deeper customer insights.
